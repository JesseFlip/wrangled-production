WrangLED

Conference lights you control from Discord.

How to play

Once you're in, drop these in any channel the bot is watching. Tap a card to copy it.

Commands are paced — one every ~7 seconds — so everyone gets a turn.

How it works

A Discord bot listens for commands and hands them to a FastAPI hub. The hub routes each command over a persistent WebSocket to a Raspberry Pi sitting next to the LED matrix, which talks to WLED over the local network. 

Discord ──▶ FastAPI ──▶ Raspberry Pi ──▶ WLED matrix
