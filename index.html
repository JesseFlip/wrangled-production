Hardware & Wiring Guide

The fastest, safest path to assemble the physical hardware in Dallas. Focus on common grounds and power safety.

Crucial Steps:

  1. Unplug AC before touching terminal screws.
  2. Wire AC Cord to PSU: Black to L, White to N, Green to ⎘ Ground.
  3. Run a Red wire from PSU V+ to Matrix 5V wire.
  4. Run a Black wire from PSU V- to Matrix GND wire.
  5. Common Ground: Run a Red wire from PSU V+ to SP803E VCC, and Black from PSU V- to SP803E GND. Controller and Matrix must share the PSU ground.
  6. Connect SP803E Data (GPIO 16) to the Matrix DIN (Data In) wire.

⚠ Hardware Pitfalls to Avoid

  • Level Shifting (Flicker Issue): ESP32s output 3.3V data. WS2812Bs want 5V data. If the matrix flickers crazily, the first LED isn't reading the 3.3V. Fix: Keep the data wire from controller to matrix under 4 inches.
  • Voltage Drop (Pink/Yellow whites): 256 LEDs at full white draw ~15A. If the end of the matrix turns pinkish, inject 5V/GND from the PSU to the other end of the matrix. (Or cap brightness in WLED to 50%).
  • Directionality: Ensure data connects to DIN, not DOUT. Look for arrows on the PCB.

WLED JSON API (HTTP POST)

Fastest way to control WLED. Send a POST request to http://[WLED-IP]/json/state.

// Example Payload to turn ON, set Red, effect "Sweep"
{
  "on": true,
  "bri": 128,         // Brightness 0-255
  "seg": [
    {
      "col": [
        [255, 0, 0],  // Primary Color (Red)
        [0, 0, 0]     // Secondary Color
      ],
      "fx": 14,       // Effect ID (14 = Sweep)
      "sx": 128       // Effect Speed
    }
  ]
}

Pi WebSocket Proxy (Jesse runs this)

A tiny Python script running on the Pi. It connects to Jim's VPS over Tailscale, waits for JSON, and HTTP POSTs it to the local WLED controller.

import asyncio
import websockets
import requests
import json

VPS_URI = "ws://[JIM_VPS_TAILSCALE_IP]:8765"
WLED_URL = "http://192.168.4.1/json/state" # Update to WLED IP

async def listen():
    async with websockets.connect(VPS_URI) as ws:
        print("Connected to VPS")
        while True:
            msg = await ws.recv()
            try:
                payload = json.loads(msg)
                requests.post(WLED_URL, json=payload)
                print(f"Sent to WLED: {payload}")
            except Exception as e:
                print(f"Error: {e}")

asyncio.run(listen())

Conference Day Protocol

PyTexas starts April 17. The network environment will be hostile (noisy WiFi). Rely on the Pixel 8a Hotspot.

⚑

Network Failsafe Configuration

Do NOT rely on venue WiFi. Configure everything for the hotspot beforehand.

  • Set Pixel 8a Hotspot SSID to PyTexas_WrangLED (5GHz preferred for less interference).
  • Hardcode this SSID/Pass into the Raspberry Pi's wpa_supplicant.conf.
  • Hardcode this SSID/Pass into WLED's WiFi settings (via WLED AP mode before the conference).
  • Ensure Pi Tailscale daemon is set to start on boot.
